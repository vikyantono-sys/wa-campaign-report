👋 Quick start
  1. Sign in with Google using your YOBO account (button below). You'll approve access once; after that it stays signed in for the session.
  2. Add a Template row for each blast — type the template name, set its Send Date range and its Trx Date range. Repeat for every template in the campaign.
  3. Click Load Outlets and tick the outlets this campaign was sent to. The same outlet set applies to every template above.
  4. Click Load from BigQuery to run the query, then scroll down and click Run Analysis.
  5. Export results with Export to Excel at the bottom of the report.
Data is queried live from BigQuery in your browser as the signed-in user — nothing is stored in the file. SQL queries are hidden by default; click Show SQL in any card to inspect or tweak.
